# データモデル: シンプル Todo アプリ

## エンティティ

### Task (タスク)

単一の ToDo 項目を表します。

| フィールド名 | 型      | 必須 | 説明                 | 備考                      |
| ------------ | ------- | ---- | -------------------- | ------------------------- |
| id           | string  | Yes  | タスクの一意な識別子 | UUID v4 を使用            |
| title        | string  | Yes  | タスクの内容         |                           |
| completed    | boolean | Yes  | 完了状態             | true: 完了, false: 未完了 |
| createdAt    | Date    | Yes  | 作成日時             | ソートに使用              |

## 関係性

- ユーザーは複数のタスクを持つ (1:N)
- _今回はシングルユーザー・ローカル保存のため、User エンティティは明示的には定義しない_

## バリデーションルール

1. **title**:

   - 必須
   - 空文字は許可しない
   - 最大文字数は設けないが、UI 上で適切に表示できること (エッジケース対応)

2. **id**:
   - 衝突しないユニークな文字列であること (`crypto.randomUUID()` 使用推奨)
