# クイックスタート & 検証ガイド: シンプル Todo アプリ

## 概要

このガイドでは、実装された Todo アプリの動作確認手順を説明します。

## 開発環境の起動

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` (または表示された URL) にアクセスします。

## 検証シナリオ

### シナリオ 1: タスクの追加と表示

1. アプリを開く。
2. 上部の入力フォームに「テストタスク」と入力する。
3. 「追加」ボタンを押すか、Enter キーを押す。
4. **確認**: 下部のリストに「テストタスク」が表示されること。
5. **確認**: 入力フォームが空になっていること。

### シナリオ 2: タスクの完了と未完了

1. リストの「テストタスク」の横にあるチェックボックスをクリックする。
2. **確認**: テキストに取り消し線が表示されること。
3. もう一度チェックボックスをクリックする。
4. **確認**: 取り消し線が消えること。

### シナリオ 3: データの永続化

1. タスクがある状態でブラウザのタブを閉じる (またはリロードする)。
2. 再度アプリを開く。
3. **確認**: 前回のタスクが同じ状態で残っていること。

### シナリオ 4: タスクの削除

1. タスクの横にある削除ボタン（ゴミ箱アイコンなど）をクリックする。
2. **確認**: そのタスクがリストから消えること。
